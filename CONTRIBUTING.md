# Contributing

Thanks for helping improve Open Source Portfolio.

## Local Setup

```bash
npm install
npm run dev
```

Before opening a pull request, run:

```bash
npm run lint
npm run build
```

## Contribution Ideas

- Improve accessibility.
- Add more portfolio sections.
- Add localization.
- Improve GitHub Pages deployment notes.
- Add tests for future interactive features.

## Pull Request Checklist

- Keep changes focused.
- Include screenshots for visual changes.
- Do not commit secrets, private links, cookies, or personal tokens.
- Update `README.md` when behavior or setup steps change.

## Style

- Prefer simple React components.
- Keep content easy to edit.
- Use semantic HTML first.
- Keep animations subtle and useful.
